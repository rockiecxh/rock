# Rock Live2D typecho plugin

## 特性 Feature
在网页中添加Live2D看板娘。与PJAX兼容，支持无刷新加载。  
Add Live2D widget to web page. Compatible with PJAX.

基于stevenjoezhang的[live2d-widget]（https://github.com/stevenjoezhang/live2d-widget）修改而成的简单插件。
